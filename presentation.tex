\subsection{Staging \& Commiting}
\begin{frame}{Staging}
Git gaat een map op je computer volgen en tracken wat er gewijzigd wordt. Het doet dit echter enkel voor bestanden die jij vastlegt.\\

Dit klaarzetten om te 'tracken' heet \textbf{staging}. Het alternatief is een bestand in de map negeren door het in het '\textit{.gitignore}' bestand te zetten.
\end{frame}
